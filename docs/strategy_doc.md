【一、核心逻辑】
  每周从 3 个维度打分，选出综合得分最高的 5 只ETF。

  得分权重分配：
    momentum     ██████████░░░░░░░░░░ 50%
    volume       ██████░░░░░░░░░░░░░░ 30%

────────────────────────────────────────────────────────────
【二、打分因子详解（共3个）】

  因子1：MomentumFactor
    → 动量因子：看最近N个交易日涨了多少。涨得多的说明资金在涌入，动量越强代表短期趋势越强。但也要小心涨太多已经透支了。
    → 计算：计算方式：最新收盘价相比最近10个交易日前涨了多少
    → 好的情况：动量值为正且越高越好，说明近期强势上涨
    → 坏的情况：动量值为负，说明近期在下跌，不选

  因子2：TrendFactor
    → 趋势因子：看价格是否在均线之上。站在均线上说明是上升趋势，跌破均线说明可能转弱。均线多头排列（短>中>长）最强。
    → 计算：计算方式：最新价格是否在20日均线之上
    → 好的情况：价格在均线以上，趋势向上
    → 坏的情况：价格在均线以下，趋势向下，不选

  因子3：VolumeFactor
    → 量能因子：看最近N天的平均成交量是过去M天平均的多少倍。量比>1表示放量，说明有资金在参与；量比<1表示缩量，可能观望情绪浓。
    → 计算：计算方式：最近5天均量 / 过去20天平均均量，结果就是'量比'
    → 好的情况：量比>1.2，近期明显放量，有资金推动
    → 坏的情况：量比<0.8，缩量，缺少资金关注

────────────────────────────────────────────────────────────
【三、剔除规则（共3个，全部通过才保留）】

  规则1：TrendFilter [✅ 启用]
    → 趋势筛选：必须站在20日均线上才保留，跌破均线的直接淘汰。
    → 具体要求：只保留价格在均线上方的ETF

  规则2：MomentumFilter [✅ 启用]
    → 动量排名筛选：只保留动量排名前N%的ETF，淘汰弱势的。
    → 具体要求：只保留动量排名前40%的ETF，比如24只里只留前9只

  规则3：VolumeFilter [✅ 启用]
    → 量能筛选：量比低于阈值的直接淘汰。量能是趋势的燃料，没有量能支撑的上涨难以持续。
    → 具体要求：量比量比>1.1（温和放量）才保留，否则淘汰

────────────────────────────────────────────────────────────
【四、最终输出】
  经过打分 + 筛选后，每次选出综合得分最高的 5 只ETF作为推荐。
  每只ETF最高仓位：25%（即最多花 25% 的钱买1只）
  最高总仓位：80%（留 20% 现金备用）

────────────────────────────────────────────────────────────
【五、止损规则】
  触发以下任一条件就考虑卖出：
  - 亏损超过 8% 时止损
  - 价格跌破20日均线时退出
  - 持仓ETF掉出当前排名前5时换仓

────────────────────────────────────────────────────────────
【六、调仓频率】
  每周调仓一次。
  注意：调仓有手续费，频繁调仓可能吃掉利润。
